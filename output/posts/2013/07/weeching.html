New perl script release, WEEChing! (Source)

This nifty, little perl script generates an I Ching hexagram, with changing lines, and then displays a reading for the hexagram, the relevant lines and the resulting hexagram. The readings are provided courtesy of deoxy.org and are retrieved with the cURL utility, parsed by string splitting and embedded into an HTML page that the user is redirected to. This is the first full version of the program, although  it has been in development for some time now. The code itself still needs to be thoroughly cleaned & further organized, but the functionality is there with no bugs witnessed in  the present version. Please feel free to report any requests or issues encountered at any time by email, comment or otherwise.

NOTE: Deoxy.org does have its own random I Ching reading here
